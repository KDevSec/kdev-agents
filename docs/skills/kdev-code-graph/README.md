## 核心功能

| 功能 | 说明 |
|------|------|
| **需求追溯** | 从设计文档(.md/图片)追溯到代码实现，带信心评分 |
| **爆炸半径分析** | 修改代码时分析影响范围，可视化依赖链 |
| **文档-代码同步** | 检查文档与代码的一致性，发现脱节问题 |
