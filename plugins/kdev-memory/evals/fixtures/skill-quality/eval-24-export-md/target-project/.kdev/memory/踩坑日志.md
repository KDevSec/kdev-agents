# 踩坑日志

## G-001

日期：2026-05-10
triggers: [文件权限, Write 工具, .kdev, 落盘失败, permission denied]

写入 `.kdev/memory/` 时遭遇文件权限问题，首次 Write 调用失败。
原因：目录未预先创建，父目录不存在导致写入失败。
解法：先用 Bash mkdir -p 建好目录再 Write。

## G-003

日期：2026-05-12
triggers: [subject 推断, F-NNN, skill-feedback, 归属错误, 分类错误]
