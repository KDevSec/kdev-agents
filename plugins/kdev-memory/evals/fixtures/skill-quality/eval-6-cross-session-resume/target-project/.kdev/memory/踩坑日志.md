# 踩坑日志

## G-014: Redis XADD 的 MAXLEN 参数位置很微妙
triggers: ["redis XADD", "MAXLEN", "stream"]
日期：2026-04-21
现象：`XADD key MAXLEN 1000 * field value` 报 syntax error
原因：~ 符号位置——`XADD key MAXLEN ~ 1000 * field value`
解决：加 ~ 号允许近似裁剪，Redis 实际按块裁剪更快

## G-015: EventSource 浏览器端 retry 默认 3 秒太短
triggers: ["EventSource", "retry", "reconnect"]
日期：2026-04-20
现象：心跳漏一次就触发重连，拖垮服务
原因：浏览器默认 retry 3 秒
解决：服务端在 headers 里发 `retry: 15000` 指令延长
