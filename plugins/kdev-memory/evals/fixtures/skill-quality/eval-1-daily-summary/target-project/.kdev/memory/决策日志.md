# 决策日志

## Q-007: JWT 库选用 PyJWT 还是 authlib？
日期：2026-04-22
选项：
- (A) PyJWT —— 简单、轻量，社区使用面广
- (B) authlib —— 功能全，含 OAuth/OIDC，未来可扩展
用户选择：B
理由：半年内要上 OIDC，不想再换库

## Q-008: refresh token 过期时间定多久？
日期：2026-04-22
选项：
- (A) 7 天 —— 业内常见默认
- (B) 30 天 —— 用户体验好，但撤销窗口长
- (C) 14 天 + 滑动续期
用户选择：C
理由：单人项目不需要严格的撤销窗口，但连续使用不希望频繁登录
