# 踩坑日志

## G-004: SQLAlchemy 2.0 AsyncSession 不能跨 event loop 复用
triggers: ["sqlalchemy", "asyncsession", "event loop"]
日期：2026-04-20
现象：线上偶发 `attached to a different loop`
原因：uvicorn worker 重用 session 对象
解决：每个请求新建 session，autouse fixture 清理

## G-005: authlib JWT 刷新 token 的 kid 字段必填否则 verify 失败
triggers: ["jwt", "kid", "authlib", "verify"]
日期：2026-04-22
现象：verify_jwt 在 refresh 流程中报 `InvalidKey: kid missing`
原因：默认生成 token 时不带 kid，但 refresh 复用的 JWKS 流程强制校验 kid
解决：在 encode 时显式传 kid=key_id，或关闭 require_kid

## G-006: pytest-asyncio 0.23 默认 scope 从 function 改成 session
triggers: ["pytest-asyncio", "fixture scope"]
日期：2026-04-22
现象：旧 fixture 跨测试污染状态
原因：升级 0.23 后默认 scope 变了
解决：显式声明 `@pytest.fixture(scope="function")` 或 rollback 到 0.21
