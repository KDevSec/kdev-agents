# 踩坑日志

## G-012: pnpm install 在 workspace 根目录会漏装子包依赖
triggers: ["pnpm install", "pnpm i", "workspace 依赖"]
日期：2026-04-15
现象：子包 import 报错 "Cannot find module 'xxx'"，package.json 里明明写了
原因：pnpm 默认只装 root，需要 --recursive 或在子包目录单独跑
解决：统一在子包里跑 `pnpm install`，或 root 跑 `pnpm -r install`

## G-014: aiohttp 代理连接 ClientDisconnectedError 高并发下崩溃
triggers: ["aiohttp", "proxy", "ClientDisconnected"]
日期：2026-04-17
现象：高并发下 proxy 经常抛 ClientDisconnectedError
原因：默认 keepalive_timeout 太短，TCP 连接频繁重建
解决：server.py:42 的 handler 包 try/except ClientDisconnectedError，并把 keepalive_timeout 调到 75
