# Claude Code 主要功能列表

根据官方文档，Claude Code 是 Anthropic 官方的 AI 编程助手工具，运行在终端环境中。以下是其主要功能和特性：

## 1. 核心定位
- **Agentic 编程工具**: 不是简单的代码补全工具，而是能够理解代码库、执行多步骤任务的智能代理
- **终端集成**: 直接在终端、IDE、桌面应用和浏览器中使用

## 2. 文件操作工具 (File Operations)
- 读取文件 (Read)
- 编辑文件 (Edit)
- 创建文件 (Write)
- 重命名文件
- 多文件编辑能力

## 3. 搜索工具 (Search)
- **Glob**: 按模式查找文件
- **Grep**: 使用正则表达式搜索文件内容
- 代码库搜索和导航

## 4. 执行工具 (Execution)
- 运行 Shell 命令 (Bash)
- 启动服务器
- 运行测试
- Git 工作流操作
- 执行构建任务

## 5. Web 工具 (Web)
- **WebSearch**: 网络搜索
- **WebFetch**: 获取网页内容
- 查阅文档和错误信息

## 6. 代码智能 (Code Intelligence)
- 检测类型错误
- 跳转到定义
- 查找引用
- 理解代码结构

## 7. 扩展功能
- **Skills (技能)**: 工作流程自动化
- **MCP (Model Context Protocol)**: 与外部服务和工具集成
- **Hooks (钩子)**: 自动化触发器
- **Subagents (子代理)**: 任务委托和并行处理
- **Plugins (插件)**: 自定义扩展

## 8. Git 工作流
- 提交代码
- 创建分支
- 代码审查
- 合并操作
- 版本控制管理

## 9. 其他核心能力
- 编写文档
- 代码审查和质量检查
- 调试和错误修复
- 多步骤工作流编排
- 交互式用户问答 (AskUserQuestion)
- 监控后台脚本 (Monitor)

## 10. Agent SDK
- 提供 API 访问 Claude Code 所有功能
- 可构建自定义代理
- 支持企业级安全和策略部署

## 与 Copilot 的区别
- Copilot 主要提供行级代码自动补全
- Claude Code 是完整的代理系统，能够读取文件、执行命令、进行多文件编辑
